# bg3-osiris

This in-development Visual Studio Code extension provides language support for Osiris, the scripting language used by Baldur's Gate 3 and other Divinity Engine games.

## Features

Planned features include:
- Syntax highlighting
- Diagnostics for syntax and program logic
- Hover support
- Autocomplete
- Story outline visualization and manipulation
- Rename symbol support
- Find all references support
